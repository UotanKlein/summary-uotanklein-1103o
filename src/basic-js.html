    • Типы данных, свойства типов, "исключения", методы

      Примитивные типы

      Undefined

      Тип: undefined
      Свойство: Означает, что переменная была объявлена, но не инициализирована.
      Исключения: Нет.
      Методы: Нет.

      Null

      Тип: object (это исключение, поскольку null не является настоящим объектом)
      Свойство: Представляет собой "ничего" или "пустое значение".
      Исключения: typeof null === "object" является исторической особенностью JavaScript.
      Методы: Нет.

      Boolean

      Типы: true или false
      Свойство: Используется для представления истины или лжи.
      Исключения: Нет.
      Методы: Boolean(value) может быть использован для преобразования значения в булев тип.

      Number

      Тип: number
      Свойства: Может представлять как целые числа, так и числа с плавающей точкой.
      Исключения: NaN (Not-a-Number) также принадлежит к типу number.
      Методы: isNaN(value), isFinite(value), Number.isInteger(value), Number.parseFloat(value), Number.parseInt(value)

      BigInt

      Тип: bigint
      Свойство: Позволяет представлять целые числа произвольной величины.
      Исключения: Нет.
      Методы: BigInt(value)

      String

      Тип: string
      Свойства: Представляет текстовые данные.
      Исключения: Нет.
      Методы: .length, .charAt(index), .indexOf(substring), .toUpperCase(), .toLowerCase(), .substring(indexStart, indexEnd)

      Symbol

      Тип: symbol
      Свойства: Представляет уникальный идентификатор.
      Исключения: Нет.
      Методы: Symbol(description)

      Объекты

      Object

      Свойства: Может содержать набор пар ключ-значение.
      Исключения: Нет.
      Методы: Object.keys(obj), Object.values(obj), Object.entries(obj), Object.hasOwnProperty(prop)

      Array

      Свойства: Представляет собой упорядоченный список элементов.
      Исключения: Несмотря на то, что массивы в JavaScript являются объектами, для них существует специальный синтаксис и методы.
      Методы: .length, .push(item), .pop(), .shift(), .unshift(item), .indexOf(item), .slice(start, end)

      Function

      Свойства: Объекты, исполняющие код.
      Исключения: Функции в JavaScript являются объектами первого класса.
      Методы: .call(thisArg, arg1, arg2, ...), .apply(thisArg, [argsArray]), .bind(thisArg)
      Date, RegExp, Map, Set и другие встроенные объекты также имеют свои уникальные свойства и методы.

      Замечания:

      Преобразование типов: JavaScript автоматически преобразует типы при необходимости (например, при использовании операторов сравнения).
      Методы проверки типов: Хотя JavaScript не предоставляет

    • Структуры данных

      Массивы (Array)

      Массивы в JavaScript — это упорядоченные списки элементов, которые могут хранить данные любого типа. Массивы являются объектами и предоставляют методы для работы с элементами.

      Свойства и методы:

      .length — возвращает количество элементов в массиве.
      .push(element) — добавляет элемент в конец массива.
      .pop() — удаляет последний элемент из массива и возвращает его.
      .shift() — удаляет первый элемент из массива и возвращает его.
      .unshift(element) — добавляет элемент в начало массива.
      .indexOf(element), .findIndex(callback), .find(callback) — методы для поиска элементов в массиве.
      .slice(start, end), .splice(index, howMany, ...items) — методы для работы с подмассивами.

      Стек (Stack)

      Стек — это абстрактная структура данных, работающая по принципу LIFO (Last In, First Out — последним пришёл, первым ушёл). В JavaScript стек можно реализовать с помощью массива.

      Операции:

      Push: Добавление элемента в вершину стека.
      Pop: Удаление и возвращение верхнего элемента стека.
      Peek: Возвращение верхнего элемента без его удаления.
      isEmpty: Проверка стека на пустоту.

      Очередь (Queue)

      Очередь — абстрактная структура данных, работающая по принципу FIFO (First In, First Out — первым пришёл, первым ушёл). В JavaScript очередь также можно реализовать с помощью массива, но для более эффективной работы часто используются специализированные подходы.

      Операции:

      Enqueue: Добавление элемента в конец очереди.
      Dequeue: Удаление и возвращение первого элемента очереди.
      Front: Возвращение первого элемента без его удаления.
      isEmpty: Проверка очереди на пустоту.

    • Арифметические операции

      Сложение (+):

      Складывает два числа.
      Пример: 5 + 3 равно 8.
      Также используется для конкатенации строк.

      Вычитание (-):

      Вычитает второе число из первого.
      Пример: 5 - 3 равно 2.

      Умножение (*):

      Умножает два числа.
      Пример: 5 * 3 равно 15.

      Деление (/):

      Делит первое число на второе.
      Пример: 6 / 3 равно 2.

      Остаток от деления (%):

      Возвращает остаток от деления первого числа на второе.
      Пример: 5 % 2 равно 1.

      Инкремент (++):

      Увеличивает значение переменной на 1.
      Если используется как префикс (++x), возвращает новое значение переменной.
      Если используется как постфикс (x++), возвращает старое значение переменной, а затем увеличивает его на 1.

      Декремент (--):

      Уменьшает значение переменной на 1.
      Если используется как префикс (--x), возвращает новое значение переменной.
      Если используется как постфикс (x--), возвращает старое значение переменной, а затем уменьшает его на 1.

      Унарный минус (-):

      Меняет знак числа на противоположный.
      Пример: -5 равно -5, если исходное число было 5.

      Унарный плюс (+):

      Преобразует операнд в число.
      Пример: Если x равно "5", +x преобразует "5" в 5.

      Возведение в степень (**):

      Возводит число в указанную степень.
      Пример: 2 ** 3 равно 8.

    • Функции библиотеки Math

      Свойства

      Math.PI: Возвращает значение числа π (пи).
      Math.E: Возвращает значение числа e, основания натуральных логарифмов.

      Функции

      Math.abs(x): Возвращает абсолютное значение числа x.
      Math.ceil(x): Возвращает наименьшее целое число, большее или равное x.
      Math.floor(x): Возвращает наибольшее целое число, меньшее или равное x.
      Math.round(x): Округляет число до ближайшего целого.
      Math.max(x1, x2, ..., xn): Возвращает наибольшее из переданных чисел.
      Math.min(x1, x2, ..., xn): Возвращает наименьшее из переданных чисел.
      Math.sqrt(x): Возвращает квадратный корень числа x.
      Math.random(): Возвращает псевдослучайное число от 0 (включительно) до 1 (исключая).
      Math.log(x): Возвращает натуральный логарифм (по основанию e) числа x.
      Math.sin(x), Math.cos(x), Math.tan(x): Возвращают соответственно синус, косинус и тангенс угла x (в радианах).
      Math.asin(x), Math.acos(x), Math.atan(x): Возвращают арксинус, арккосинус и арктангенс x, соответственно.

    • Логические операции

      И (&&)

      Операция "И" возвращает true, если оба операнда истинны (т.е., оба значения равны true).
      Пример: true && true возвращает true; true && false возвращает false.
      В контексте булевых преобразований: если первый операнд может быть преобразован в false, возвращается первый операнд; в противном случае возвращается второй операнд.

      ИЛИ (||)

      Операция "ИЛИ" возвращает true, если хотя бы один из операндов истинен.
      Пример: true || false возвращает true; false || false возвращает false.
      В контексте булевых преобразований: если первый операнд может быть преобразован в true, возвращается первый операнд; в противном случае возвращается второй операнд.

      НЕ (!)

      Операция "НЕ" инвертирует булево значение операнда: true становится false, и наоборот.
      Пример: !true возвращает false; !false возвращает true.
      В контексте булевых преобразований: операнд преобразуется к булевому типу, а затем значение инвертируется.

      Нулевой оператор слияния (??)

      Оператор нулевого слияния возвращает правый операнд, если левый операнд равен null или undefined, и левый операнд во всех остальных случаях.
      Пример: null ?? 'default' возвращает 'default'; 'value' ?? 'default' возвращает 'value'.

      Оператор строгого сравнения (=== и !==)

      Строгое равенство (===) проверяет равенство без приведения типов, т.е., возвращает true только если оба операнда имеют одинаковый тип и значение.
      Строгое неравенство (!==) является противоположностью строгого равенства и проверяет, не равны ли операнды, также без приведения типов.

      Оператор нестрогого сравнения (== и !=)

      Нестрогое равенство (==) проверяет равенство с приведением типов, т.е., возвращает true, если операнды равны после приведения их к общему типу.
      Нестрогое неравенство (!=) является противоположностью нестрогого равенства и проверяет, не равны ли операнды после приведения их к общему типу.

    • Алгоритмические конструкции

      Последовательность

      Последовательность — это простейшая алгоритмическая конструкция, представляющая собой выполнение инструкций одна за другой в заданном порядке. Каждая инструкция выполняется точно один раз, в порядке их следования в программе.

      Ветвление (Условные операторы)

      Ветвление позволяет программе выбирать между двумя или более путями выполнения в зависимости от истинности заданных условий. Основные конструкции ветвления:
      if: Выполняет блок кода, если условие истинно.
      if-else: Выполняет один блок кода, если условие истинно, и другой блок кода, если условие ложно.
      switch-case: Позволяет выбирать между множеством блоков кода на основе значения переменной или выражения.

      Циклы

      Циклы используются для повторения блока кода несколько раз. Это позволяет программе выполнять повторяющиеся задачи эффективно. Основные типы циклов:
      for: Повторяет блок кода определённое количество раз.
      while: Продолжает выполнять блок кода, пока условие истинно.
      do-while: Выполняет блок кода один раз, а затем продолжает его выполнение, пока условие истинно.

      Функции (подпрограммы)

      Функции позволяют группировать блоки кода, которые выполняют определённую задачу. Это способствует повторному использованию кода, улучшает структурированность и читаемость программы. Функции могут принимать параметры и возвращать результат.

      Рекурсия

      Рекурсия — это частный случай функций, когда функция вызывает сама себя. Это мощная алгоритмическая конструкция, позволяющая решать сложные задачи более простым способом, но требует аккуратного определения условия выхода из рекурсии, чтобы избежать бесконечного цикла.
      Примеры использования

      Последовательность:

      let x = 5;
      let y = x + 2;
      console.log(y);

      Ветвление:

      if (x > 5) {
      console.log("x больше 5");
      } else {
      console.log("x меньше или равно 5");
      }

      Циклы:

      for (let i = 0; i < 5; i++) {
      console.log(i);
      }
      let i = 0;
      while (i < 5) {
      console.log(i);
      i++;
      }

      Функции:

      function sum(a, b) {
      return a + b;
      }
      console.log(sum(5, 3));

      Рекурсия:

      function factorial(n) {
      if (n === 0) {
      return 1;
      } else {
      return n * factorial(n - 1);
      }
      }
      console.log(factorial(5));

    • Функции

      Объявление функции

      Объявление функции (или Function Declaration) позволяет определить функцию с использованием ключевого слова function.
      function greet(name) {
      return `Hello, ${name}!`;
      }
      console.log(greet("Alice")); // "Hello, Alice!"
      Этот способ объявления функции поддерживает hoisting, то есть функцию можно вызвать до её объявления в коде.

      Функциональное выражение

      Функциональное выражение (или Function Expression) определяет функцию внутри выражения, часто присваивая её переменной. Функциональные выражения могут быть анонимными или именованными.
      const greet = function(name) {
      return `Hello, ${name}!`;
      };
      console.log(greet("Bob")); // "Hello, Bob!"

      Стрелочные функции

      Стрелочные функции представляют собой сокращённый синтаксис для создания функций и особенно полезны для анонимных функций. Они не имеют собственного контекста this.
      const greet = (name) => `Hello, ${name}!`;
      console.log(greet("Charlie")); // "Hello, Charlie!"
      Стрелочные функции могут быть более компактными, если функция принимает только один аргумент и выполняет одно выражение:
      const square = x => x * x;
      console.log(square(4)); // 16

      Импорт и экспорт функций

      В современном JavaScript для модулей используются ключевые слова import и export для обмена функциями и другими сущностями между файлами. Экспорт функции
      // file: math.js
      export function sum(x, y) {
      return x + y;
      }

      Импорт функции

      // file: app.js
      import { sum } from './math.js';
      console.log(sum(1, 2)); // 3
      Также можно использовать экспорт по умолчанию, когда предполагается экспортировать одну основную сущность из файла:
      // Экспорт
      export default function greet(name) {
      return `Hello, ${name}!`;
      }
      // Импорт
      import greet from './greet.js';
